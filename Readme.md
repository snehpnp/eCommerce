## 🧩 Overview

`SmartStore` is a full-featured e-commerce platform built using the **MERN** stack. It supports multiple user roles including:

- 👤 **Admin**
- 🧑‍💼 **Seller**
- 🛍️ **User**

This system handles **product management**, **user authentication**, **secure logins**, and **image uploads** using Cloudinary.

---

## 🚀 Features

✅ Role-based access (Admin, Seller, User)  
✅ Login via **SSO (Google)** and **Normal Email/Password**  
✅ Dynamic product listing with sorting, category filtering, and search  
✅ Toggle between **Card View** 🧾 and **Table View** 📊  
✅ Image upload for products/logos using **Cloudinary**  
✅ Responsive design with Bootstrap & React

## 🛠️ Tech Stack

| Tech        | Usage         |
|-------------|----------------|
| ⚛️ React     | Frontend UI |
| 🌿 Node.js & Express | Backend APIs |
| 🍃 MongoDB   | NoSQL Database |
| ☁️ Cloudinary | Image Uploads |
| 🔐 JWT Auth | Secure Login System |
| 🧠 Redux     | State Management |
| 🌐 SSO (Google) | Single Sign-On Login |

---

## 🔑 Roles & Permissions

| Role   | Permissions |
|--------|-------------|
| 🛠️ Admin  | Add/edit/delete users, manage products |
| 🧑‍💼 Seller | Add/edit/delete own products |
| 🛍️ User   | Browse, search, and purchase products |
